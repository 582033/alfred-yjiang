## 适用于alfred workflows的工具集

* go开发
* * 基于[awgo](https://github.com/deanishe/awgo)
*
*
* ## 功能列表:
*
* #### 时间/时间戳互转; example:
*
* ```
* yjiang now
* yjiang 1535414400
* yjiang 2018-12-12 12:12:12
* ```
*
*
* #### ip归属地查询; example
*
* > 从 `http://ip-api.com/`获取ip归属地信息
*
* ```
* yjiang 114.114.114.114
* ```
*
